Maze Game Site Plan

Kaden Saltzgiver | WDD 131-Fall 2024

Overview

Purpose: The purpose of this site is to provide a fun and interactive maze game where users can navigate through a maze and track their time. The game includes features like a timer, collision detection with walls, and different levels with degrees of difficulty.

Audience: Casual gamers or anyone wanting a quick challenge or to test their mouse skills.

Dynamic elements:

  • A real-time timer that tracks the user's time spent in the maze.
  • Collision detection with walls, where if the user touches a wall, an alert is shown, and the timer pauses.
  • A mouse-tracking feature that detects if the user leaves the page and shows an alert.
  • A dynamic game-over and completed maze modal that shows alerts depending on the player's actions.
  • A function to load maze information from a JSON file.

Content

Home page: The home page will introduce the maze game, explain its objective (get from start to end without touching walls), and include a start button to begin playing the game. A timer will be visible, and the page will have a retro-style design with simple navigation.

Level Select: This page will be a simple navigation to the desired level.

Leaderboard: This page will be a leaderboard for the best times.

Instructions: This page will include the instructions for how to play the game, how to interact with the maze, and how to use the Esc key to return to the main menu.
